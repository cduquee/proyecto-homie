La prueba de Kolomogorov-Smirnov es la siguiente:

Prueba de Hipotesis

\[H_0: f_X(x) = f_0(x)\] \[H_A: f_X(x) \neq f_0(x)\] \[D = \max_{1\leq i \leq N}{\left\{\frac{i}{n}-F_X(x_i), \ F_X(x_i)-\frac{i-1}{n}\right\}}\]
